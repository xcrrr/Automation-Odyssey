# SEO Content Engine

SEO最適化コンテンツ生成エンジン。キーワード調査→競合分析→最適化記事生成。

## 収益ポテンシャル

- **単価**: $100-300/記事
- **生産量**: 5-10記事/日
- **月収目安**: $500-3,000/月

## ワークフロー

```
1. キーワード調査
   ├── メインキーワード分析
   ├── 関連キーワード抽出
   └── 検索ボリューム・競合度確認

2. 競合分析
   ├── 上位10サイトの構造分析
   ├── 見出し構成の抽出
   └── 不足トピックの特定

3. コンテンツ生成
   ├── SEO最適化タイトル生成
   ├── メタディスクリプション作成
   ├── H2/H3構造化見出し
   ├── 本文執筆（1,500-3,000語）
   └── 内部リンク提案

4. 品質チェック
   ├── キーワード密度確認（1-2%）
   ├── 読みやすさスコア
   └── 重複コンテンツチェック
```

## 使用方法

### 基本的な記事作成
```
「"best coffee makers 2026"でSEO記事を書いて」
「"ダイエット 簡単"でブログ記事作成」
```

### 詳細指定
```
「キーワード: [keyword]
 ターゲット: [audience]
 文字数: [word_count]
 トーン: [professional/casual/friendly]」
```
